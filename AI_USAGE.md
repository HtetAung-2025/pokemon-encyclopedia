### プロンプト5: CSSアニメーションの相談

```

```

**AIの対応**:

- bounceアニメーションの実装例

アプリの技術流れ
1. ホームページを開く
2. useEffect が実行される
3. loading = true になり、ローディング画面が表示
4. fetch でPokeAPIからデータを取得
5. データが返ってきたら setPokemon(data) で保存
6. setLoading(false) でローディング終了
7. ポケモンカードが表示される
